  1. Explain inheritance along with its types.

    it is a mechanism in which one object acquires all the properties & behaviours of a parent objet
    The main advantage is reusability and the disadvantag is that the parent and class are tightly coupled.
    any changes made in the parent class will be reflected in all the corresponding classes.
    inheritance is one of the cornerstones of object oriented programming because it allows the creation of hierarchy classifications.
    in the Java terminology, a class that is inherited that is called a superclass and the class that inherits is called a subclass.
    a subclass inherits all the members defined by superclass and adds its own unique elements.
    the general form of a class declaration that inherits a superclass, as shown here.
    class subclass name extends superclass name
    {
    // body of class.
    }
    types of inheritance are :-
    1) single inheritance- a class inherits the properties of a single superclass.
    a->b
    2) multi level inheritance- when a class extends a class that extends another class.
    a->b->c
    3) hierarchal inheritance- multiple child classes inherit the methods of the same parent class.
    *diagram
    4) multiple inheritance- subclass inherits more than one superclass.
    *diagram
    5) hybrid inheritance- combination of multiple and single inheritance.
    *diagram

    1. Foreach loop.
      in Java for each is used to iterate through elements of arrays and collections, also known as Enhanced for loop.
      syntax - for(datatype item : array)
      {
      //body
      }

      eg - sum of all elements of an array using for each loop.
      class main
      {
      public static void main(String[] args)
      {
      int[] numbers = {3,4,5,6,-5,0,12};
      int sum = 0;
      for (int number : numbers)
      {
      sum += number;
      }
      System.out.println("sum = "+sum);
      }
      output. -> sum = 25.


    2. Break.

      public class BreakExample {
      public static void main(String[] args) {
      for (int i = 1; i <= 10; i++) {
      if (i == 5) {
      break;
      }
      System.out.println(i);
      }
      System.out.println("Loop stopped at 5.");
      }
      }


    3. Continue.

      public class ContinueExample {
      public static void main(String[] args) {
      for (int i = 1; i <= 10; i++) {
      if (i == 5) {
      continue;
      }
      System.out.println(i);
      }
      System.out.println("Loop continued at 5.");
      }
      }


    4. return

      public class ReturnExample {
      public static void main(String[] args) {
      for (int i = 1; i <= 10; i++) {
      if (i == 5) {
      return;
      }
      System.out.println(i);
      }
      System.out.println("Loop ended at 5.");
      }
      }
