-A single value which can define the entire data is called measure of central tendency.

  1. Mathematical average : using mathematical formulae we can calculate the average.
    there are 3 methods to calculate the average :

Arithematic mean for ungrouped data


let X takes the values x1,x2......xn
then the arithematic mean is defined by :-
 X̄ = (∑x)/n ; where n = number of x


Ex1. find the avg. of 10 employees salary
the salary are : 25000, 22500, 24000, 28000, 32500, 40000, 33000, 34600, 37400, 42500
- sol
here n = 10
X̄ = 320000/10 = 32000

Arithematic mean for grouped data :

1) Discrete data:

Let X take the values x1,x2......xn with the corresponding frequency f1,f2......fn then,
to find the arithematic mean we use,
 X̄ = (∑fx)/n ;
where n is the sum of frequency.
