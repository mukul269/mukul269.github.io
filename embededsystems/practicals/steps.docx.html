On Keil:

  • Project->new project-> open your folder and name it(file).
  • Search for AT89C51->OK->NO
  • Click on target 1->Source group
  • File->new->program->save it with .c
  •  Right click Source group->add existing files->select your file
  • Right click target->options->xtal->11.0592->output tab->create hex file
  • Right click target-> build target
  • Check in object folder for hex file

On Proteus:

  • New project->name->next->next->finish
  • Pick device option(2nd one in yellow)->select AT89C51->Select LED any colour
  • Select and place devices
  • From terminals select ground
  • Click on microcontroller chip->select program file(HEX)->change frequency=11.0592
