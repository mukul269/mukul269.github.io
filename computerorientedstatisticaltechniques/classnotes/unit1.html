A---------median-----------B

  1. if n = odd
    then median = ((n+1)/2) th term

Median for grouped data :
step 1 - find the cumulative frequency of less than type, then select the mmiddle class i.e. the 1st exceeding the cumulative frequency to the value N/2.
The corresponding class is called middle class.
for discrete data middle class value is the median.
for continuous data to calculate median we use the formulae

median = l 1 + ((N/2 - cf)/f) * (l2-l1)

Where, l1 = lower limit of middle class interval;
l2 = upper limit of middle class interval;
f = frequency of middle class;
cf = cumulative frequency of pre-middle class
